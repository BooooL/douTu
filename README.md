# douTu 斗图神器

<img src="screenshot/doutu.webp" title="Logo"  width="300"  alt="Demo"/>

## 原理  
  
- curl请求 斗图网站  
- gumbo 爬虫解析  
- qt5 UI展示  

## 依赖  
  
- curl  
- gumbo  
- Qt5  
  
## 编译  
  
````shell script
$ cd douTu
$ mkdir build && cd build
$ cmake -DCMAKE_BUILD_TYPE=Release ..
$ make
````
